Approach (วิธี IRB) ให้สาขาของธนาคารพาณิชย์ต่างประเทศนับผลต่างดังกล่าวเป็นเงินสํารองทั่วไป
(General provision) เพื่อนำไปคำนวณเงินสำรองที่กันไว้แล้ว (Total eligible provisions) ได้ตาม

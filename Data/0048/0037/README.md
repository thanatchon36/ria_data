สามสิบวันทําการ นับแต่วันที่ได้รับความเห็นจากธนาคารแห่งประเทศไทยตามวรรคสอง เว้นแต่
มีเหตุจําเป็นไม่อาจพิจารณาให้แล้วเสร็จภายในระยะเวลาดังกล่าว ให้รัฐมนตรีมีหนังสือแจ้งให้

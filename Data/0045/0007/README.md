ผู้ประกอบธุรกิจต้องพิจารณาถึงความมั่นคงและฐานะทางการเงิน

หรือกิจการดังกล่าวเป็นรายบุคคล ทั้งนี้ ผู้ประกอบธุรกิจต้องดูแลมิให้มีการยื่นคำขอมีบัตรเครดิตเพื่อธุรกิจ

ประกันก่อนหน้าวันที่หลักเกณฑ์ที่เกี่ยวข้องกับสินเชื่อที่มี
ทะเบียนรถเป็นประกันมีผลบังคับใช้ และประสงค์
ประกอบธุรกิจดังกล่าว :
เป็นประกัน

ก่อนการเริ่มประกอบธุรกิจสินเชื่อที่มีทะเบียนรถ
